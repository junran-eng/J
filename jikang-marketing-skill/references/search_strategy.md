# 搜索策略指南

各内容类型的最优搜索策略，指导如何高效获取外部信息为推文提供素材。

## 搜索原则

1. **先精后广**：先用精准关键词攻核心信息，再用宽泛词补背景
2. **时效优先**：优先抓取近6个月内发布的内容
3. **权威来源**：政府网站 > 行业标准 > 学术文献 > 行业媒体 > 自媒体
4. **交叉验证**：关键数据至少从2个独立来源确认

## 事件类型搜索策略

### 搜索目标
- 确认事件的基本信息（时间、地点、议程、参与方）
- 了解同类事件的行业背景和以往情况
- 收集可供引用的行业观点和数据

### 搜索流程

```
第1轮（核心信息）："[事件名称] 2026 时间 地点" → 确认5W1H
第2轮（背景信息）："[事件领域] 行业趋势 2026" → 获取宏观背景
第3轮（对比信息）："往届[事件名称] 回顾 亮点" → 建立纵深视角
第4轮（补充信息）："[关键词] 最新动态 2026" → 补充时效性素材
```

### 优质来源
- 吉康官网 (https://www.gdjikang.com/) — 公司新闻、产品动态
- 广东省环境保护产业协会 (http://www.gdepi.com/) — 广东环保产业活动
- 中国环保产业协会 (http://www.caepi.org.cn/) — 全国性行业展会、论坛
- 行业媒体（中国水网、E20环境平台、北极星环保网）
- 参展企业官方新闻稿

## 技术类型搜索策略

### 搜索目标
- 获取技术原理的权威解释
- 收集可量化的性能参数和数据
- 寻找真实的应用案例和运行数据
- 了解技术发展趋势和前沿动态

### 搜索流程

```
第1轮（技术原理）："[技术关键词] 原理 工艺 流程" → 建立技术框架
第2轮（数据参数）："[技术关键词] 能耗 效率 含水率 对比" → 收集核心数据
第3轮（应用案例）："[技术关键词] 应用案例 工程实例 运行数据" → 寻找真实案例
第4轮（前沿趋势）："[技术领域] 最新进展 2025 2026 创新" → 捕捉前沿信息
第5轮（选型参考）："[设备类型] 选型 参数对比 注意事项" → 实用参考信息
```

### 搜索关键词库（低温除湿干化）
- 核心技术：热泵干燥、低温蒸发、闭式循环、余热回收
- 关键参数：含水率、除湿量、COP、单位能耗、处理能力
- 对比框架：vs 板框压滤、vs 带式干燥、vs 桨叶干燥、vs 太阳能干化
- 应用场景：市政污泥、印染污泥、制药污泥、餐厨垃圾

### 优质来源
- 吉康官网 (https://www.gdjikang.com/) — 产品参数、技术规格、应用案例
- 中国节能协会热泵专委会 (https://www.chpa.org.cn/) — 热泵技术前沿、行业标准
- 中国知网（cnki.net）学术论文
- 专利数据库（国家知识产权局）
- 行业技术白皮书
- 环保技术论坛（水世界、水网论坛等）

## 政策类型搜索策略

### 搜索目标
- 获取政策原文
- 寻找权威的政策解读
- 了解行业反应和影响分析
- 收集合规建议和应对方案

### 搜索流程

```
第1轮（政策原文）："site:gov.cn [政策关键词]" → 定位官方原文
第2轮（官方解读）："[政策名称] 解读 政策吹风会" → 获取官方立场
第3轮（行业影响）："[政策关键词] 对污泥处理行业 影响" → 分析行业影响
第4轮（应对方案）："污泥处理 环保合规 技术方案 标准" → 收集应对建议
第5轮（关联政策）："[政策领域] 相关法规 标准 配套文件" → 完善政策图景
```

### 重点关注的政府网站
- 生态环境部 (https://www.mee.gov.cn/) — 环保法规、排放标准、污染防治政策
- 工业和信息化部 (https://www.miit.gov.cn/) — 产业政策、制造业规划、节能环保装备政策
- 国家能源局 (https://www.nea.gov.cn/) — 能源政策、节能规划、余热利用政策
- 中国政府网 (gov.cn)
- 住房和城乡建设部 (mohurd.gov.cn)
- 国家发展和改革委员会 (ndrc.gov.cn)
- 国家标准化管理委员会 (sac.gov.cn)
- 各省生态环境厅

### 行业协会来源
- 广东省环境保护产业协会 (http://www.gdepi.com/) — 地方政策解读、行业动态
- 中国环保产业协会 (http://www.caepi.org.cn/) — 全国性政策解读、行业标准
- 中国节能协会热泵专委会 (https://www.chpa.org.cn/) — 热泵行业政策与技术规范

### 优质来源
- 政府官方发布（法规、标准、通知）
- 国务院政策例行吹风会解读
- 行业权威媒体政策栏目
- 协会政策解读文件
- 专业律所/咨询机构政策分析报告

## 搜索关键词优化技巧

### 时间限定
- 最新信息：`"[关键词] 2026"`、`"[关键词] 2025"`
- 近两年：`"[关键词] 2025 2026"`
- 未来趋势：`"[关键词] 趋势 展望 预测"`

### 网站限定
- 政府网站：`site:gov.cn [关键词]`
- 学术资源：`site:cnki.net [关键词]`
- 行业媒体：`site:h2o-china.com [关键词]`

### 精确匹配
- 完整名称：`"[政策全称]"`
- 文件号：`"[政策文号]"`

### 排除干扰
- `[关键词] -推广 -广告 -招聘`
- `[关键词] -价格 -报价`（技术搜索时排除销售信息）

## 信息质量评估清单

| 评估维度 | 高分特征 | 低分特征 |
|---------|---------|---------|
| 时效性 | 6个月以内 | 2年以上 |
| 权威性 | 政府/学术/行业权威 | 个人博客/自媒体 |
| 完整性 | 有数据、有来源、有方法 | 仅有观点、无数据支撑 |
| 客观性 | 多方观点，承认局限性 | 一面倒、明显营销倾向 |
| 相关性 | 直接切题 | 勉强沾边 |
