# 内容类型写作策略

吉康环境公众号推文三大内容类型的详细写作策略。

## 一、事件类推文

**目标**：通过公司动态和行业活动展示吉康的行业参与度和专业形象。

### 适用场景
- 行业展会参展预告/回顾（中国环博会、上海水展等）
- 重大项目签约/竣工/投产仪式
- 客户考察来访
- 公司重大新闻（新产品发布、资质获取、获奖等）
- 行业论坛/峰会参与

### 写作框架

```
导语（1-2句）→ 事件详情（5W1H）→ 吉康角色（我们做了什么/展示了什么）
→ 行业意义（为什么重要）→ 行动号召（下一步）
```

### 写作要点
- **时效性优先**：事件类推文生命期短（3-5天），需要快速发布
- **图片来源**：现场照片 > 产品图 > 资料图（配图建议写入内容）
- **情感基调**：积极、自信，展现公司活力和行业地位
- **避免**：流水账式记录、过于自我宣传、缺少行业视角

### 示例标题
- 「吉康环境亮相2026中国环博会，低温除湿干化方案引关注」
- 「签约！吉康环境助力XX污水厂污泥减量化项目」
- 「XX市领导莅临吉康环境考察指导，共话污泥处理新路径」

### 字数建议：1200-1800字

---

## 二、技术类推文

**目标**：建立专业权威，教育市场，帮助客户理解技术价值并做出采购决策。

### 适用场景
- 低温除湿干化技术原理解析
- 设备选型指南/参数对比
- 应用案例深度分析
- 行业痛点与技术解决方案
- 新技术/新工艺发布
- 常见问题FAQ

### 写作框架

```
痛点引入（Problem）→ 问题放大（Agitate）→ 技术原理（核心价值）
→ 吉康方案（Solve）→ 应用案例（信任背书）→ 选型建议（促成行动）
```

### 写作要点
- **数据驱动**：能耗数据、含水率变化、投资回报周期等需要具体数字
- **通俗解释**：技术术语首次出现需解释，善用类比（如"像空调除湿原理一样"）
- **对比展示**：与传统方案（热干化、板框压滤等）的参数对比表
- **避免**：过度夸大技术优势、引用未验证数据、缺少实际案例支撑

### 关键技术话题库
1. 低温除湿干化 vs 传统热干化：能耗、温度、适用性对比
2. 污泥含水率从80%降到30%意味着什么？
3. 热泵技术在污泥干化中的应用原理
4. 如何根据污泥性质选择干化设备？
5. 低温干化在市政/工业/印染等不同场景的应用差异

### 字数建议：1800-2500字

## 三、政策类推文

**目标**：帮助客户理解政策影响，展示吉康的合规能力和前瞻性。

### 适用场景
- 国家/地方环保新标准发布
- 行业政策法规解读
- 政策趋势分析
- 环保督查/专项整治动态
- 行业规划文件解读

### 写作框架

```
政策速览（3句话get核心）→ 政策背景（为什么出台）→ 关键条款解读（对行业的影响）
→ 影响分析（短期+长期）→ 吉康应对（我们如何满足/超越要求）
→ 行动建议（给客户的时间表和checklist）
```

### 写作要点
- **准确引用**：政策名称、文号、发布单位、实施日期必须准确
- **务实解读**：不仅说"是什么"，更要说"对你有什么影响"
- **建设性**：不只说"压力来了"，还要说"怎么办"
- **时效窗口**：政策类推文有最佳发布期（政策发布后1-2周内）

### 重点政策清单（持续更新）
1. 《污泥处理处置污染防治技术政策》
2. 《城镇污水处理厂污泥处理技术规程》
3. 各省市污泥处理处置规划
4. "双碳"目标下污泥处理政策导向
5. 环保督察中污泥处理典型案例

### 字数建议：1500-2200字
